# monaco-editor-docs
monaco-editor 中文文档

肝力有限，求点赞

这是个人在使用Monaco-editor时，感觉到官网的查看缺少例子，所以采用了文档比较详细的vue文档风格进行整理。

在线地址：[在线地址](https://aydk-xcc.github.io/monaco-editor-docs/)

完善中ing

## 1. 安装

``` shell
pnpm install
```

## 2. 运行
``` shell
pnpm docs:dev

// http://localhost:8081/monaco-editor-docs/
```

## 3.打包

``` shell
pnpm docs:build
```
