# CommentRule

## 属性 Properties

### blockComment
+ 类型: [CharacterPair](../alias.md#characterpair) `可选`
+ 默认值: `-`
+ 可选项: `-`
+ 描述: `块注释字符对，如 /* 块注释 */`

### lineComment
+ 类型: `string`  `可选`
+ 默认值: `-`
+ 可选项: `-`
+ 描述: `行注释标记，例如 // 这是一条注释`
