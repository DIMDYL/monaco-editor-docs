# IEditorCommentsOptions
`interface` [官网地址](https://microsoft.github.io/monaco-editor/docs.html#interfaces/editor.IEditorCommentsOptions.html)

## 属性 Properties

### ignoreEmptyLines?
+ 类型: `boolean`  `可选`
+ 默认值: `true`
+ 可选项: `-`
+ 描述: `插入行注释时忽略空行。默认为true。`

### insertSpace?
+ 类型: `boolean`  `可选`
+ 默认值: `true`
+ 可选项: `-`
+ 描述: `在行注释标记之后和块注释标记内插入一个空格。默认为true。`
