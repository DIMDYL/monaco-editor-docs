# IPasteAsOptions
`interface` [官网地址](https://microsoft.github.io/monaco-editor/docs.html#interfaces/editor.IPasteAsOptions.html)

## 属性 Properties

### enabled?
+ 类型: `boolean`  `可选`
+ 默认值: `true`
+ 可选项: `-`
+ 描述: `是否启用粘贴功能`

### showPasteSelector?
+ 类型: `enum`  `可选`
+ 默认值: `afterPaste`
+ 可选项: `"never" | "afterPaste"`
+ 描述: `控制是否在粘贴之后显示粘贴的提示组件`
