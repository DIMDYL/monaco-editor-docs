# INewScrollPosition
`interface` [官网地址](https://microsoft.github.io/monaco-editor/docs.html#interfaces/editor.INewScrollPosition.html)
用于描述编辑器的滚动位置。
## 属性 Properties
### scrollLeft?
+ 类型: `number` `可选`
+ 默认值: `-`
+ 可选项: `-`
+ 描述: `水平滚动位置。 `

### scrollTop?
+ 类型: `number`  `可选`
+ 默认值: `-`
+ 可选项: `-`
+ 描述: `垂直滚动位置。 `
