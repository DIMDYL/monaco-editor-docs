# IDropIntoEditorOptions
`interface` [官网地址](https://microsoft.github.io/monaco-editor/docs.html#interfaces/editor.IDropIntoEditorOptions.html)


## 属性 Properties
### enabled
+ 类型: `boolean`  `可选`
+ 默认值: `true`
+ 可选项: `-`
+ 描述: `启用拖放到编辑器中。默认为true。`

### showDropSelector
+ 类型: `enum`  `可选`
+ 默认值: `afterDrop`
+ 可选项: `"never" | "afterDrop"`
+ 描述: `控制放置后是否显示小部件。默认为“afterDrop”。`
