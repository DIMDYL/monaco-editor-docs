# IEditorStickyScrollOptions
`interface` [官网地址](https://microsoft.github.io/monaco-editor/docs.html#interfaces/editor.IEditorStickyScrollOptions.html)
## 属性 Properties

### defaultModel?
+ 类型: `enum`  `可选`
+ 默认值: `-`
+ 可选项: `"outlineModel" | "foldingProviderModel" | "indentationModel"`
+ 描述: `默认情况下为粘性滚动选择的模型`

### enabled?
+ 类型: `boolean` `可选`
+ 默认值: `-`
+ 可选项: `-`
+ 描述: `指定是否允许粘性滚动`


### maxLineCount?
+ 类型: `number`  `可选`
+ 默认值: `-`
+ 可选项: `-`
+ 描述: `显示的最大粘性行数`
